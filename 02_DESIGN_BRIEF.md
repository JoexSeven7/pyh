# MEDPLUS PHARMACY - DESIGN BRIEF

## Emmanuel Keshi Branch, Magodo/Ketu, Lagos

---

## I. BRAND OVERVIEW

**Business Name:** Medplus Pharmacy (Emmanuel Keshi Branch)
**Location:** 32 Emmanuel Keshi St, Ikosi Ketu, Lagos 105102 (Magodo/Ketu area)
**Primary USP:** 24-Hour Pharmacy with Delivery & In-Store Pickup
**Target Audience:** Professionals, families, seniors seeking convenient, reliable healthcare solutions in Magodo/Ketu
**Brand Voice:** Professional, trustworthy, accessible, community-focused, approachable

---

## II. COLOR PALETTE

### Primary Colors

- **Pharmacy Blue (Primary):** #0066CC or #004DA6
  - Conveys trust, health, professionalism
  - Use for: Main CTA buttons, headers, brand elements
- **Clinical White (Background):** #FFFFFF or #F8FAFB

  - Clean, medical aesthetic
  - Use for: Main background, readability

- **Wellness Green (Accent):** #2DBE60 or #27AE4F
  - Signifies health, vitality, wellness
  - Use for: Secondary buttons, success messages, highlights

### Secondary Colors

- **Soft Gray (Text/Borders):** #333333, #666666, #CCCCCC
- **Alert Red (Emergency/Urgent):** #E74C3C (for "Call Now" buttons only)
- **Success Green:** #27AE4F (order confirmations, positive feedback)

### Color Usage Guidelines

```
Header/Navigation: Pharmacy Blue background with white text
Hero CTA Buttons:
  - "Order Delivery" → Wellness Green
  - "Call Now" → Alert Red (urgency for emergencies)
Trust Section: Light background with Pharmacy Blue text
Map Section: Clean, minimal borders
Text: Dark gray (#333) on white for 4.5:1 contrast ratio
Links: Pharmacy Blue, underline on hover
```

## III. TYPOGRAPHY

### Font Selection

**Primary Font (Headlines & CTAs):**

- Poppins (Bold, SemiBold) - Modern, clean, healthcare-appropriate
- Alternative: Inter, Montserrat

**Secondary Font (Body Text):**

- Poppins Regular (400) or Open Sans (400, 500)
- Clean, highly readable on mobile
- 16px minimum for body text

### Type Scale

```
H1 (Hero Headline):    48px - 64px (Desktop), 32px (Mobile) | Bold
H2 (Section Headers):  32px - 40px (Desktop), 24px (Mobile) | SemiBold
H3 (Subheaders):       24px - 28px (Desktop), 20px (Mobile) | SemiBold
Body Text:             16px - 18px | Regular (400)
Small Text/Labels:     14px | Regular
Button Text:           16px | SemiBold
```

### Font Weight Usage

- Bold (700): Headlines, CTAs, emphasis
- SemiBold (600): Subheaders, important labels
- Regular (400): Body copy, descriptions
- Light (300): Secondary info, timestamps

### Overall Look

- **Minimalist + Professional**
- Clean white space, minimal clutter
- High-quality photography (pharmacy interior, products)
- Icon-driven for key services

---

## VII. ACCESSIBILITY REQUIREMENTS

1. **Color Contrast:** Minimum 4.5:1 ratio (WCAG AA standard)

   - All text readable against backgrounds
   - CTA buttons have sufficient contrast

2. **Typography:**

   - Minimum 16px for body text
   - Line height: 1.5 - 1.6 for readability
   - Line length: 50-75 characters max

3. **Navigation:**

   - Keyboard accessible (Tab, Enter)
   - Focus states clearly visible
   - Skip navigation link (hidden but available to screen readers)

4. **Images:**

   - All images have descriptive alt text
   - Decorative images: empty alt attribute (alt="")

5. **Forms:**

   - Label associated with each input
   - Clear error messages
   - Success feedback provided

6. **Interactive Elements:**
   - Minimum 48px touch target size (mobile)
   - Clear focus indicators
   - Adequate spacing between clickable elements

---

## VIII. RESPONSIVE DESIGN STRATEGY

### Desktop (1200px+)

- Two-column layouts where appropriate
- Horizontal navigation bar
- Full-width hero image
- Side-by-side testimonial display (optional)

### Tablet (768px - 1199px)

- Responsive grid adjustments
- Hamburger menu optional
- Single-column for most content
- Adjusted font sizes

### Mobile (<768px)

- Single-column layout
- Full-width buttons
- Hamburger menu (required)
- Vertical spacing increased for touch
- Images optimized for mobile
- Text size increased
- Stack testimonials vertically

## IX. ANIMATIONS & INTERACTIONS

### Subtle Animations

- **Button Hover:** Color change + slight shadow increase (0.2s transition)
- **Link Hover:** Color change + underline (0.15s)
- **Testimonial Slide:** 0.5s transition between slides
- **Form Focus:** Light blue glow around input (0.2s transition)
- **Scroll Animations:** Fade-in effects for below-fold content (optional, not overused)

### Interaction Feedback

- **Click Feedback:** Button scales slightly (0.95x) on click
- **Form Submission:** Loading spinner, then success message
- **WhatsApp Click:** Opens WhatsApp (web or app)
- **Phone Click:** Initiates call on mobile

## XI. PERFORMANCE REQUIREMENTS

### Image Optimization

- Hero image: Compressed, WebP format, responsive sizes
- Icons: SVG format (scalable, small file size)
- Service images: Optimized JPEGs, lazy-loaded

### File Size Targets

- Hero image: < 200KB (compressed)
- Total CSS: < 50KB
- Total JS: < 100KB (minimal framework)
- Page load: < 3 seconds on 4G

### SEO Considerations

- Meta descriptions: 155-160 characters
- H1 tags: One per page, contains primary keyword
- Schema markup: LocalBusiness schema for address/phone/hours
- Mobile-first indexing: Fully responsive design
- Open Graph tags: For social sharing
