# Medplus Pharmacy - Image Assets

This directory contains all image assets for the Medplus Pharmacy website.

## Required Images

The following images need to be added to this directory:

### Hero Images

- `hero-bg.jpg` - High-quality pharmacy interior photo (1920x1080px minimum)
- `hero-bg-mobile.jpg` - Mobile-optimized hero image (768x1024px)

### Service Icons

- `24-hours.svg` - Clock icon for 24/7 service
- `prescription.svg` - Prescription icon
- `beauty.svg` - Spa/beauty icon
- `equipment.svg` - Medical equipment icon

### Testimonial Images (Optional)

- `testimonial-1.jpg` - Customer photo
- `testimonial-2.jpg` - Customer photo
- `testimonial-3.jpg` - Customer photo

### Favicon Assets

- `favicon.ico` - 32x32px favicon
- `apple-touch-icon.png` - 180x180px Apple touch icon
- `favicon-16x16.png` - 16x16px favicon
- `favicon-32x32.png` - 32x32px favicon

### Social Media Images

- `social-share.jpg` - Image for social media sharing (1200x630px)

## Image Specifications

### Hero Images

- Format: JPEG or WebP
- Quality: 80-90%
- Compression: Optimized for web
- Alt text: "Medplus Pharmacy interior - clean, professional, well-organized"

### Icons

- Format: SVG (preferred) or PNG
- Size: 48px - 64px for service icons
- Color: Use CSS for coloring, keep SVGs monochrome

### Favicon

- Format: ICO for favicon.ico, PNG for others
- Sizes: 16x16, 32x32, 180x180px
- Background: Transparent or white

## Image Optimization

All images should be:

- Compressed for web use
- Named with descriptive, SEO-friendly names
- Sized appropriately for their usage
- Have proper alt text for accessibility

## Placeholder Images

While final images are being prepared, you can use:

- Unsplash images for hero backgrounds
- FontAwesome icons for service icons
- Generated placeholder images for testimonials
